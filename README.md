###### MODO DE USO DEL SIMULADOR

1. Se debe escribir un nombre y un apellido válido (sin tildes ni números).
2. Se debe ingresar una materia válida, comprendida entre:

- Ingles.
- Matematica.
- Biologia.
- Quimica.
- Programacion.

3. Se debe ingresar la nota obtenida en la materia (comprendida entre 1 - 10).

**Luego se obtendrá el promedio final, dividiendo las notas de las materias dividio la cantidad de materias.**
